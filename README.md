# Next.js Project

This project uses [Next.js](https://nextjs.org/) with [Drizzle ORM](https://orm.drizzle.team/).

## Installation

```bash
npm install
# or
yarn install
# or
pnpm install
# or
bun install
```

## Environment

Duplicate `.env.example` to `.env` and fill in the values.

## Database

```bash
drizle-kit generate
drizle-kit migrate
drizle-kit push
```

## Development

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun run dev
```

## Build

```bash
npm run build
# or
yarn build
# or
pnpm build
# or
bun run build
```

## Start

```bash
npm run start
# or
yarn start
# or
pnpm start
# or
bun run start
```
